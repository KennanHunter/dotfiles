# Dotfiles

Welcome to the official Kennan Hunter dotfile repo

## Installation process

`cp ./vim/.vimrc ~/.vimrc`

`cp ./zsh/.zshrc ~/.zshrc`

`cp ./i3 ./kitty ./alacritty ./polybar ./wal ~/.config`

`cp ./wallpaper-scripts ~/bin`

## Dependencies

- zsh
- alacritty
- kitty
- vim
- i3-gaps
- polybar
- pywal
- pywal-discord
- pywal-fox
- betterlockscreen
