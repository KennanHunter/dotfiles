## Installation process

`cp ./vim/.vimrc ~/.vimrc`

`cp ./zsh/.zshrc ~/.zshrc`

`cp ./x/.xinitrc ~/.xinitrc`

`cp -r ./i3 ./kitty ./alacritty ./polybar ./wal ./rofi ./conky ./neofetch ~/.config`

`cp -r ./wallpaper-scripts ~/bin`
